# Customer Management API

REST API för kundhantering byggt med Flask och SQLite.

## Stack

- Flask 2.3.0
- PyJWT 2.6.0
- Werkzeug 2.3.0
- SQLite

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Endpoints

| Method | Path      | Beskrivning        |
|--------|-----------|--------------------|
| GET    | /health   | Hälsokontroll      |
